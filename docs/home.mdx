## Lina Framework

Lina is an extensible, un-opinionated graphql powered Nodejs backend with a suite of features and apps to support most of the basic and advance features your project will need. Built with typescript, Lina brings statically typed Javascript foremost.
<br/>

## Where does Lina fit into ?

Lina is a cross breed of fullstack and backend framework. Lina hopes to replaces frameworks like Express, etc.

<br/>


## Lina Alternatives

Non specific for now. Some similar however more different frameworks are MeteorJs, VulcanJs, NextJs, etc.
